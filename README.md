To test or modify the project locally, follow these steps:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) and a package manager for Node like [pnpm](https://pnpm.io/) installed on your machine.


### 3. Install dependencies

### 4. Start the development server
```bash

```
